# login-uploading
Website dimana kamu bisa mengirim, menyimpan, membagi project kamu dalam bentuk zip/rar
<pre>
<p>Setting config/database.php sesuai username dan password kamu</p>
<p>Import login-upload(2) kedalam database kamu</p>
</pre>
